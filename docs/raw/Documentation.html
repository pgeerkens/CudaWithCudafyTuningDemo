The code for this project was built using C# in Visual Studio Professional 2010, and makes use of:

  • CUDA 5.0 from NVidia (https://developer.nvidia.com/cuda-downloads)
  • CUDAfy 4.0 (http://cudafy.codeplex.com/discussions)
  • C++ Express 2010
  • DOT NET Framework 4.0

Performance measurements were obtained on a HP Pavilion dv6 with NVidia GE-Force GT 630M graphics card, using the software listed above.

  • Complete results for 11-cities
  • Complete Results for 12 Cities

The DemonstratePermutations project verifies that the alternative implementation of FindPathFromRoutePermutation actually covers all permutations properly.

The class MyCUDAfy.cs contains generic extension methods for the class GPGPU that provide compile-time checking of the arguments for GPGPU.Launch:

public static void SafeLaunch<T1[, ..., T15]>(this GPGPU @this, 
   dim3 gridSize, dim3 blockSize, 
   Action<GThread,T1[, ..., T15]> action, 
   T1 t1[, ..., T15 t15]);
